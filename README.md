# Day Brightness

Автоматическая регулировка яркости экрана ноутбука на основе данных с веб-камеры и положения солнца.

## Требования

* **ОС:** Ubuntu 22.04+ (или любой дистрибутив на базе Debian).
* **Железо:** Проверено на видеокартах `Intel`, необходима веб-камера.
* **Сеть:** Интернет требуется 1 раз в год (для обновления таблицы солнца).

---

## Установка

1. **Скачайте проект:**
   ```bash
   git clone https://github.com
   cd day-brightness
   ```

2. **Соберите пакет:**
   ```bash
   ./DEBIAN/build.sh
   ```

3. **Установите его в систему:**
   ```bash
   sudo apt install ./releases/day-brightness.deb
   ```
   *Система сама скачает все нужные зависимости (Python, OpenCV и утилиты подсветки).*

---

## Управление

Скрипт работает автоматически в фоне и обновляет яркость каждую минуту. Для управления используются стандартные команды:

* **Проверить, работает ли автозапуск:**
  ```bash
  systemctl status day-brightness.timer
  ```
* **Посмотреть логи работы скрипта:**
  ```bash
  sudo journalctl -u day-brightness.service -f
  ```

---

## Удаление

Полное удаление программы и всех её настроек из системы:

```bash
sudo apt purge day-brightness
```
